BetterFinance App (Flask, Bootstrap & SQLite)

BetterFinance is a robust web application designed for logged-in users to engage in various financial activities. Users have the ability to "buy & sell" stocks, "deposit & withdraw" demo money, and update or reset their login details. The application fetches real-time stock quotes from the IEX API, ensuring accurate data for informed decision-making. Additionally, users can conveniently monitor their stock portfolio and review transaction history to stay on top of their financial activities.

BetterBirthday App (Flask, Bootstrap & SQLite)

A web application is designed to help users organize birthdays. With this tool, users can effortlessly add, edit, and delete birthday entries. The interface is intuitively designed for ease of use. Additionally, the application seamlessly integrates with a database to store and manage all birthday-related data. Furthermore, session management ensures that each user's data is kept separate, eliminating the need for any registration or login process.
